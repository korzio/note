Use
@oclif/cli-ux or any other tools to

  • show a spinner while loading information,
  • print the list,
  • colors for printing open & closed issues.

Use
@octokit/rest for Github requests

import Octokit = require('@octokit/rest')

const { data: issues } = await octokit.issues.listForRepo({
  owner: args.owner,
  repo: args.repo,
})

Configure your Github list command

  1. Create Personal token
  2. Add it to config file .githubrc
  3. Use auth key with @octokit/rest
  4. Get list of Github issues
